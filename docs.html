마지막 업데이트: 2025-10-17
+
+
+
+
+

개요

+

+ 이 페이지는 노션과 아무 연동 없이 노션을 연상시키는 정적 UI로 작성되었습니다. + 문단/헤딩/콜아웃/토글/체크리스트/테이블/코드블록/목차(TOC)/브레드크럼을 제공합니다. +

+
+
💡
+
내용은 HTML만 수정하면 되고, 외부 폰트·JS 라이브러리는 사용하지 않습니다.
+
+
+ +
+

작성 지침

+
+ 섹션 추가 방법 +

아래 템플릿을 복제해 id만 바꾸고, 우측 목차(#tocNav)에 앵커를 추가하세요.

+ 
<section id="new-section">
+  <h2>새 섹션</h2>
+  <p>내용...</p>
+</section>
+
+
+ 캐시 우회 +

정적 배포 후 캐시가 남아 있으면 쿼리 파라미터를 붙여 새로고침합니다.

+ 
/docs.html?v=20251017
+/index.html?v=20251017a
