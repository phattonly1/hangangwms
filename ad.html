2. 시스템 이용방법

가. 일지작성

  • 일반항목, 총유기탄소, 질산성질소, 암모니아성질소, 유류
    : 점검내역 및 교정 시 필수 입력, 표준물질 측정값(0.0/0.0) 형식으로 입력
  • 채수시설 : 4개 항목 전부 필수체크
  • 전송장비 : 3개 항목 전부 필수체크
  • 항온항습기 : 작동상태 필수체크
  • 상태변경 발생원인 : 점검/운전정지 사유 등 입력

※ 여과시설, UPS 설치측정소 없음

※ 화면 캡처와 입력 예시는 원본 문서의 이미지 부분 참고

나. 운전정지 등록/해제

  • 운영관리 > 운전정지 > 등록에서 등록하기
  • 운전정지 시간 및 항목 입력, 사유 입력

다. 자료선별

  • 시범사업 > 자료선별 확정 > 데이터 선별 클릭
  • 측정소 및 검색기간 설정 후 조회버튼 클릭
  • 신규차트 클릭 후 캡처(☆필수)
  • 분석조건 설정 후 선별작업 수행

※ 화면 예시는 원본 문서의 차트 및 캡처 그림 참고

분석조건 설정방법

항목명 분석조건 기본값 설정이유
수소이온농도 최소치 4, 최대치 11 - 교정값 초과 및 비정상적인 수치 제외
용존산소 최소치 1, 최대치 20 - 비정상적인 수치 제외
전기전도도 설정 X - 하천에 따라 유동적으로 판단
수온 설정 X - 여름철 저온, 겨울철 고온 등 비정상적인 수치 발생 시
탁도 최소치 0.1 - 0값 제외
총유기탄소 최소치 0.05 - 0값 제외
질산성질소 최소치 0.05 - 0값 제외
암모니아성질소 설정 X - 측정소에 따라 유동적으로 판단
유류 1차선별 대상 아님 - -
  • 시범사업 > 자료선별 확정 > 유효데이터 확정 클릭
  • 헌팅 등 비정상값이 있으면 수정항목 선택에서 항목 체크 후 수정상태 정보를 ‘비정상’으로 수정하고 상태정보 변경 클릭
  • 반대로 코드 미변경으로 날아간 데이터는 ‘정상’ 변경

※ 변경 후 수정 날짜가 초기화되기 때문에 여러 개 변경 시 변경날짜 확인 필수

  • 수정 완료 시 자료확정완료 클릭

※ 자료확정 완료 후 다시 수정하고 싶으면 자료확정취소 클릭

자료선별 Tip(신규차트의 데이터 패턴)

  1. 전기전도도 확인
    전기전도도가 0에 가깝게 되는 기간은 채수가 안 된 것으로 보아 전 항목 데이터를 해당 기간 동안 제외
  2. 수소이온농도/탁도 확인
    값이 헌팅했던 경우 점검 후 안정화기간인 경우 전 항목 데이터를 몇 시간 더 제외.
    안정화기간이 아닌 경우, 값이 천천히 내려오면(좌측) 데이터 제외 X, 나타났다 사라지면(우측) 제외
  3. 용존산소 확인
    교정 후 측정값 구간이 바뀌는 경우 교정이 잘 되었는지 다음에 확인 필요(제외 안 해도 됨).
    그래프가 상하 반복이 아니라 일정값 지속 시 센서 확인 필요(사유에 따라 제외 가능)
  4. 총유기탄소/질산성질소/암모니아성질소 확인
    값이 0과 최대값에서 홀딩된 경우, 0과 최대값을 왔다 갔다 하는 경우 제외(자동선별 시 제외됨)

※ 각 항목의 그래프 패턴 예시는 원본 문서의 차트 이미지 참고

3. 보고서 작성

가. 자료 구성

  • 붙임1 : 현황 요약, 정도관리 현황, 비정상가동 내역(조별 담당자 작성 필요)
  • 붙임2 : 측정소 별 일간 가동률(조별 담당자 작성 필요)
  • 붙임3 : 차트 그래프 모음(총괄 담당자 작성)

나. 붙임 1 작업(매월 초 조장에게 메일 발송)

  • 보정현황 입력(교정/표준물질 측정 시 해당 날짜 기입)

다. 붙임 2 작업

  • 선별완료 시 유효데이터 조회에서 1일 데이터 엑셀 출력
  • 붙임2 파일에 붙여넣고 색 표현, 메모 작성, 총일수 수정

※ 가동율, 유효측정일수의 함수 확인(30일 ↔ 31일 바뀌는 경우 함수 사라짐)

색 표현방법

색 구분 구분명 내용
(예: 흰색) 정상데이터 유효측정일수 포함
(예: 노랑) 유효데이터 제외 유효측정일수 제외(가동율 저하)
(예: 파랑) 운전정지 총일수 제외(가동율 영향 없음)
(예: 회색) 정도검사/일반시험 총일수 제외(가동율 영향 없음)

메모 작성방법 및 예시

  • [측정항목] 0/00~00 ← 날짜
  • 측정불가(사유 : 00)으로 데이터 제외 다수 ← 사유(아래 참조)

날짜 예시

  • 11/1~20 ← 일반적인 경우 달마다 끊어서 작성함
  • 11/2~계속 ← 운전정지 중인 경우 시작날짜를 입력
  • 22/12/2~7/12 ← 장기 운전정지가 끝난 경우 연도를 표시해야 하는 수도 있음

사유 예시

  • 채수불가(사유 : 수위 저하)로 운전정지
  • 측정불가(사유 : 강우로 인한 간헐 채수)으로 데이터 제외 다수
  • 측정불가(사유 : 다량의 협잡물로 간헐적 채수)으로 데이터 제외 다수
  • 측정불가(사유 : 센서 이상)으로 데이터 제외 다수
  • 0값 측정(사유 : 기포 흡착)으로 데이터 제외 다수
  • 0값~최대값 반복 측정(사유 : 부유물 흡착)으로 데이터 제외 다수
  • 최대값 측정(사유 : 부유물 흡착)으로 데이터 제외 다수
